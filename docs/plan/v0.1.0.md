# CC Dashboard v0.1 実装計画

## Context

Claude Code の OpenTelemetry データを受信・可視化する個人向け軽量ダッシュボード。
Cloudflare Workers + D1 でゼロインフラの代替を構築する。

## スコープ

**やること:**
- OTLP HTTP/JSON レシーバー（logs + metrics）
- D1 への構造化保存
- Bearer token 認証
- SSR テーブルベースのダッシュボード

**やらないこと:**
- Chart.js 等のクライアントサイドチャート（v0.2）
- user_prompt / tool_decision イベント保存（v0.2）
- データ保持期間の自動削除（v0.3）
- protobuf 対応
