# ダッシュボード ダークモード化

## Context

ダッシュボードが白背景で目が辛いため、常時ダークモードに変更する。ライト/ダーク切り替え機能は不要。全6コンポーネントの Tailwind カラークラスを差し替える。

## カラーマッピング

| 要素 | 現在（ライト） | 変更後（ダーク） |
|------|---------------|-----------------|
| ページ背景 | `bg-gray-50` | `bg-gray-950` |
| ページテキスト | `text-gray-900` | `text-gray-100` |
| ヘッダー/カード/テーブル背景 | `bg-white` | `bg-gray-900` |
| テーブルヘッダー行 | `bg-gray-50` | `bg-gray-800` |
| 主ボーダー | `border-gray-200` | `border-gray-700` |
| 行区切りボーダー | `border-gray-100` | `border-gray-800` |
| ラベル/サブテキスト | `text-gray-500` | `text-gray-400` |
| 補助テキスト | `text-gray-400` | `text-gray-500` |
| 成功率（緑） | `text-green-600` | `text-green-400` |
| 成功率（黄） | `text-yellow-600` | `text-yellow-400` |
| 成功率（赤） | `text-red-600` | `text-red-400` |

### タスク 1: 全コンポーネントのカラークラスを差し替え

**変更対象**:
- `src/components/Layout.tsx` — body, header
- `src/components/Overview.tsx` — Card コンポーネント
- `src/components/DailyTokens.tsx` — テーブル
- `src/components/DailyCosts.tsx` — テーブル
- `src/components/ToolUsage.tsx` — テーブル + 成功率カラー
- `src/components/RecentSessions.tsx` — テーブル

**受入条件**:
- WHEN ダッシュボードを開く THEN 背景がダーク（gray-950）、テキストが明るい（gray-100）
- WHEN カード・テーブルを見る THEN 背景が gray-900、ボーダーが gray-700
- WHEN ToolUsage の成功率を見る THEN ダーク背景でコントラストが十分な色（400系）
